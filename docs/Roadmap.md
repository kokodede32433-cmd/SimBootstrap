# Project Roadmap — SimBootstrap

This document outlines the project roadmap and milestones.

## Sprints Schedule

### Sprint 0: Repository Foundation (Completed)
- Solution layout and CPM setup.
- Platform-safe conditional build properties.
- Empty package manifest specifications.
- Logging wrappers and empty engine interfaces.

### Sprint 1: Local Manifest Execution
- Implementation of manifest parser.
- Core downloader (HTTP/HTTPS) and verifier (File hashes, registry entries).
- CLI tool runner support.

### Sprint 2: UI Launcher & MVVM Client
- WPF user interface implementation.
- Progress monitoring bars, package selections, status dashboards.
- Rollback execution triggers.

### Sprint 3: Cloud Provisioning & SimAgent Integration
- Synchronization of local states with Club dashboard servers.
- Automatic installation triggering from remote commands.
