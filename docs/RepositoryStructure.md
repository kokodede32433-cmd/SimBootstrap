# Repository Directory Structure

This document indexes the folder layouts within the SimBootstrap repository.

```
SimBootstrap/
├── .github/                # CI workflows & issue templates
├── assets/                 # Icons and image placeholders
├── config/                 # Setup templates (Settings, Logging, Packages)
├── docs/                   # Architectural and development manuals
├── package-manifests/      # JSON package schema examples
├── src/                    # Primary source code layers
│   ├── SimBootstrap.UI/
│   ├── SimBootstrap.SDK/
│   ├── SimBootstrap.Engine/
│   ├── SimBootstrap.Core/
│   ├── SimBootstrap.Packages/
│   ├── SimBootstrap.Contracts/
│   ├── SimBootstrap.Common/
│   └── SimBootstrap.Logging/
└── tests/                  # Unit and integration test runners
```
